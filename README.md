# snapsnap

A small browser game built with plain HTML, CSS, and JavaScript.

## Run it

Open [index.html](/Users/andrewegelhof/Documents/New project/index.html) in a browser, or serve the folder with any static file server.

## Goal

Catch 5 squirrels before the 60 second round ends while a second corgi loops in front of you and knocks you off your line.

## Controls

- `WASD` or arrow keys move the player corgi
- Drag on the canvas to steer on touch screens
- `Space` or `Enter` starts or restarts the round
